# PVM for Windows

PVM (Python Version Manager) for Windows is a simple tool to manage multiple Python versions on your Windows machine.

## Installation

To install PVM for Windows, follow these steps:

1. Download the latest release from the [releases page](https://github.com/medchakkir/pvm/releases).
2. Extract the archive to a directory of your choice.
3. Add the directory to your system's PATH environment variable.

## Usage

To use PVM for Windows, open a new terminal and run the following commands:

```bash
pvm list
pvm install <version>
pvm use <version>
```
